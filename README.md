# allandabre.com

Personal portfolio for Allan Dabre — Risk & Compliance Leader at PwC, specializing in enterprise risk assessments, IT general controls, business process control design, and AI-powered compliance automation.

## Tech Stack

- **React 19** — Component architecture
- **Tailwind CSS 4** — Utility-first styling
- **Vite 7** — Build tooling & dev server
- **Cursor AI** — AI-assisted development

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

Output goes to `dist/` — deploy those files to any static hosting provider.

## Deployment

Designed for deployment on Vercel or Netlify with the included `_headers` file for production security headers (CSP, X-Frame-Options, etc.).
